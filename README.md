# Multi-Domain Price Prediction System

A production-ready machine learning dashboard built using Streamlit.

## Features

- House Price Prediction (California & Bengaluru)
- Car Price Prediction (India & Germany)
- Gold Price Forecasting
- Unified Model Comparison
- Real-time user prediction forms

## Tech Stack

- Python
- Scikit-Learn
- TensorFlow (LSTM)
- Streamlit
- Pandas / NumPy

## Deployment

Deployed using Streamlit Cloud.
